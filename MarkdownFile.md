# 📊 Financial Loan Analysis Project

## Overview

This project focuses on analyzing a financial loan dataset containing **38K+ loan records** to uncover insights related to customer behavior, loan performance, repayment trends, and risk patterns.

The project follows a complete end-to-end data analytics workflow:

**Excel → SQL → Python → Power BI**

## 🎯 Project Objectives

*   Clean and transform raw loan data
*   Perform business analysis using SQL
*   Conduct exploratory data analysis (EDA)
*   Build an interactive Power BI dashboard
*   Generate actionable business insights
*   Identify patterns related to loan risk and customer behavior

## 🛠 Tools & Technologies Used

*   **Excel** – Data cleaning & preprocessing
*   **PostgreSQL** – Business queries & analysis
*   **Python**
    *   Pandas
    *   Matplotlib
*   **Power BI** – Dashboard creation & visualization

## 📈 Key Performance Indicators (KPIs)

*   Total Loan Applications
*   Total Loan Disbursed
*   Total Repayments Received
*   Average Interest Rate
*   Average Debt-to-Income Ratio (DTI)

## 📊 Dashboard Features

### Loan Trend Analysis

*   Monthly loan application trends
*   Term-wise analysis (36 vs 60 months)

### Loan Status Analysis

*   Fully Paid
*   Charged Off
*   Current Loans

### Purpose Analysis

*   Purpose-wise funding analysis

### State Analysis

*   State-wise loan distribution

### Grade Analysis

*   Grade-based loan applications

### Income vs Loan Analysis

*   Relationship between annual income and loan amount

## 🔍 SQL Business Analysis Performed

*   Total number of loan applications
*   Total funded amount
*   Loan status distribution
*   State-wise analysis
*   Purpose-wise funding analysis
*   Grade-based risk analysis

## 📉 Python EDA

Performed:

*   Missing value analysis
*   Distribution analysis
*   Correlation analysis
*   Trend identification
*   Visual exploration

## 💡 Key Insights

*   Fully paid loans represent the majority of loan applications
*   Certain loan purposes receive significantly higher funding
*   Higher grades generally show lower risk patterns
*   Income level influences loan amount behavior
*   Loan trends vary across terms and states
